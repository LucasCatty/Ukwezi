
# Ukwezi

Ukwezi is a Kigali period-care storefront built with React and Vite.

## Running locally

Run `npm i` to install dependencies.

Run `npm run dev` to start the development server.

Run `npm run build` before deploying.

## New features

- Checkout section with MTN MoMo, Airtel Money, and bank transfer choices.
- Orders saved locally under `ukwezi_orders` for testing.
- Cleaner Kinyarwanda copy in the main translated UI.
- Header music control.
- Netlify and Vercel deployment config.

See `DEPLOYMENT.md` for database and real payment integration steps.
